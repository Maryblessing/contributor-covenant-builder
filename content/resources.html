Resources for Community Moderators

Before adopting Contributor Covenant, take the time to consider how you will enforce it. A code of conduct without safe, fair, and transparent enforcement sends a false signal and can have a disastrous impact on marginalized or vulnerable people.

These resources may be helpful in preparing to enforce your code of conduct.

  • Community Safety and Accountability
    (via Safety First PDX)

  • Enforcing Your Code of Conduct: Effective Incident Response
    (by Audrey Eschright)

  • Django Code of Conduct Enforcement Manual
    (via Django Project)

  • Code of Conduct Enforcement Materials
    (via OpenJS Foundation)

  • How We’re Making Code of Conduct Enforcement Real — and Scaling it
    (by Emma Irwin)
